LLM4SAST

LLM4SAST is an LLM-driven universal testing framework for Static Application Security Testing (SAST) tools. Basically, it includes three components: program transformation engine, dynamic feedback driven selector, and program differencing fault cluster. Until now, LLM4SAST has found 66 bugs in known SAST tools.
